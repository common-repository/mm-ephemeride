Simply display on your WordPress site: the current date, the Saints of the day and the times of sunrise and sunset. All this whith a shortcode.

== Description ==

This plugin display a simple ephemeris everywhere you need on your wordpress site. It uses shortcode that you can use on pages, posts, widgets, etc. 

Display options:
*Show / hide the date 
*Change the sentence before the date 
*Change the date display format 
*Show / hide names to celebrate 
*Show / hide sunrise and sunset times
